# Logseq Link Filter

A Logseq plugin that adds filter buttons to the **Linked References** section, allowing you to quickly toggle between:

- **All** — Show all linked references (default)
- **Journals** — Show only references from journal pages
- **Pages** — Show only references from non-journal pages
